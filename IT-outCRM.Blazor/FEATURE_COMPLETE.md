- ✅ Создание заказа (`/orders/create`)

- ✅ Фильтрация по статусам
- ✅ Пагинация
- ✅ Выбор клиента, исполнителя и статуса из списков

### 👥 Управление клиентами (Customers)
**Доступ:** Admin, Manager

- ✅ Список всех клиентов (`/customers`)
- ✅ Создание клиента (`/customers/create`)
- ✅ Редактирование клиента (`/customers/edit/{id}`)
- ✅ Детальный просмотр (`/customers/{id}`)

- ✅ Поиск по аккаунту и компании

### 📦 Мои заказы (My Orders)
**Доступ:** Все пользователи

- ✅ Персональная страница заказов (`/my-orders`)
- ✅ Статистика: всего, в работе, завершено, сумма

- ✅ Создание нового заказа
- ✅ Card-based дизайн

### 📊 Dashboard
**Доступ:** Все авторизованные пользователи

- ✅ Адаптивная статистика под роль пользователя
- ✅ Для Admin/Manager: заказы, клиенты, активные, завершенные

## 🎯 Требования к паролю

Для обеспечения безопасности пароль должен содержать:

- ✅ Минимум **8 символов**
- ✅ Заглавную букву (A-Z)
- ✅ Строчную букву (a-z)
- ✅ Цифру (0-9)
- ✅ Спецсимвол (!@#$%^&* и т.д.)

**Примеры валидных паролей:**
- `Admin@2024`
- `Manager#123`
- `MyPass123!`
- `Client!2024`

## 🔄 API Endpoints (Backend)

```
POST   /api/auth/register        - Регистрация
POST   /api/auth/login           - Вход
GET    /api/orders               - Список заказов
POST   /api/orders               - Создание заказа
GET    /api/orders/{id}          - Детали заказа
PUT    /api/orders/{id}          - Обновление заказа
DELETE /api/orders/{id}          - Удаление заказа
GET    /api/customers            - Список клиентов
POST   /api/customers            - Создание клиента
GET    /api/customers/{id}       - Детали клиента
PUT    /api/customers/{id}       - Обновление клиента
DELETE /api/customers/{id}       - Удаление клиента
GET    /api/companies            - Список компаний
POST   /api/companies            - Создание компании
DELETE /api/companies/{id}       - Удаление компании
GET    /api/accounts             - Список аккаунтов
DELETE /api/accounts/{id}        - Удаление аккаунта
GET    /api/executors            - Список исполнителей
POST   /api/executors            - Создание исполнителя
DELETE /api/executors/{id}       - Удаление исполнителя
GET    /api/order-statuses       - Список статусов заказов
```
