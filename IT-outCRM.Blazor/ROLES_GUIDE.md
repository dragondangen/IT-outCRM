# 🎭 Руководство по ролям пользователей IT-outCRM

## 📋 Обзор

CRM-система IT-outCRM теперь поддерживает **три роли пользователей** с разными уровнями доступа:

1. **👑 Администратор (Admin)** - полный доступ ко всем функциям
2. **👔 Менеджер (Manager)** - управление заказами и клиентами
3. **👤 Клиент (Client)** - просмотр только своих заказов

---

## 🔐 Распределение прав доступа

### 👑 Администратор (Admin)

**Доступные страницы:**
- ✅ Dashboard - полная статистика по системе
- ✅ Все заказы - просмотр, создание, редактирование, удаление всех заказов
- ✅ Клиенты - управление всеми клиентами
- ✅ Отчеты (скоро) - аналитика и отчеты
- ✅ Профиль - личная информация

**Навигация:**
```
- Dashboard
- Все заказы
- Клиенты
- Отчеты
- Профиль
- Выйти
```

**Особенности:**
- Видит все заказы и всех клиентов
- Может создавать, редактировать и удалять любые записи
- Доступ к административным функциям
- Полная статистика на дашборде (4 карточки: заказы, клиенты, активные, завершенные)

---

### 👔 Менеджер (Manager)

**Доступные страницы:**
- ✅ Dashboard - статистика по заказам и клиентам
- ✅ Все заказы - управление заказами
- ✅ Клиенты - управление клиентами
- ✅ Профиль - личная информация

**Навигация:**
```
- Dashboard
- Все заказы
- Клиенты
- Профиль
- Выйти
```

**Особенности:**
- Видит все заказы и всех клиентов
- Может создавать, редактировать и удалять заказы и клиентов
- Полная статистика на дашборде (4 карточки)
- Нет доступа к административным функциям

---

### 👤 Клиент (Client)

**Доступные страницы:**
- ✅ Dashboard - упрощенная статистика о своих заказах
- ✅ Мои заказы - просмотр и создание своих заказов
- ✅ Профиль - личная информация

**Навигация:**
```
- Dashboard
- Мои заказы
- Профиль
- Выйти
```

**Особенности:**
- Видит только свои заказы (в будущем будет фильтрация по CustomerName)
- Может создавать новые заказы
- Может просматривать детали своих заказов
- Упрощенная статистика на дашборде (4 карточки: мои заказы, в работе, завершено, общая сумма)
- **Не может** редактировать или удалять заказы
- **Не может** видеть других клиентов

---

## 🆕 Новые страницы

### 📝 Редактирование

#### `/orders/edit/{id}` - Редактирование заказа
- Доступно: Admin, Manager
- Функции:
  - Редактирование всех полей заказа
  - Выбор клиента из списка
  - Выбор исполнителя из списка
  - Изменение статуса
  - Обновление цены и описания

#### `/customers/edit/{id}` - Редактирование клиента
- Доступно: Admin, Manager
- Функции:
  - Изменение аккаунта
  - Изменение компании

### 👤 Профиль пользователя

#### `/profile` - Страница профиля
- Доступно: **все авторизованные пользователи**
- Отображает:
  - Имя пользователя и роль
  - Статус аутентификации
  - Права доступа в зависимости от роли
  - Информация о безопасности (скоро)
  - История активности (скоро)

**Особенности:**
- Красивый градиентный header с аватаром (инициалы)
- Цветная карточка с правами в зависимости от роли:
  - 🔴 Красная рамка - Администратор
  - 🟡 Желтая рамка - Менеджер
  - 🔵 Синяя рамка - Клиент

### 📦 Мои заказы

#### `/my-orders` - Личные заказы клиента
- Доступно: **все пользователи**
- Функции:
  - Просмотр своих заказов (для клиентов)
  - Просмотр всех заказов (для админов и менеджеров)
  - Поиск по названию или описанию
  - Фильтрация по статусу
  - Статистика: всего заказов, в работе, завершено, общая сумма
  - Создание нового заказа

**Особенности:**
- Современный card-based дизайн
- Цветные статусы заказов
- Быстрый переход к деталям заказа
- Адаптивная сетка карточек

## 🛠️ Технические детали

### Реализация ролей

```csharp
// В NavMenu.razor
@if (context.User.IsInRole("Admin"))
{
    // Показываем только для админов
}

@if (context.User.IsInRole("Admin") || context.User.IsInRole("Manager"))
{
    // Показываем для админов и менеджеров
}

// Для клиентов показываем, если НЕ админ и НЕ менеджер
@if (!context.User.IsInRole("Admin") && !context.User.IsInRole("Manager"))
{
    // Клиентский контент
}
```

### Защита страниц

```csharp
// Защита всей страницы
@attribute [Authorize]

// Защита по ролям
@attribute [Authorize(Roles = "Admin")]
@attribute [Authorize(Roles = "Admin,Manager")]
```

### Условное отображение на странице

```csharp
<AuthorizeView>
    <Authorized>
        @if (context.User.IsInRole("Admin"))
        {
            // Контент только для админов
        }
    </Authorized>
</AuthorizeView>
```

## 📊 Dashboard по ролям

### Администратор и Менеджер видят:
- 📦 Всего заказов
- 👥 Всего клиентов
- ✅ Активные заказы
- ⏳ Завершенные заказы
- Список последних заказов (5 шт.)
- Быстрые действия (создание заказа/клиента, просмотр списков)

### Клиент видит:
- 📦 Мои заказы
- 🕐 В работе
- ✅ Завершено
- 💰 Общая сумма
- Список его заказов
- Возможность создать новый заказ

## 🚀 Быстрый старт

```

Введите логин и пароль
```
